# Agropecuária no Paraná 🌱 | Agrinho 2026

Um site interativo desenvolvido como projeto para o **Programa Agrinho 2026**, destacando a força, a sustentabilidade e a tecnologia da agropecuária paranaense.

## 📖 Sobre o Projeto

Este projeto foi criado por um aluno do 1º Ano do Ensino Médio com o objetivo de participar do concurso Agrinho. O site serve como um material informativo e visual sobre o papel fundamental do campo na nossa sociedade, abordando:

* **O Gigante Paranaense:** O impacto econômico da produção de grãos e proteína animal.
* **Sustentabilidade:** Como o Paraná lidera práticas de conservação, como o Plantio Direto e a Integração Lavoura-Pecuária-Floresta (ILPF).
* **Agro 4.0:** A tecnologia de ponta aplicada no campo, incluindo drones, tratores autônomos e Internet das Coisas (IoT).

## ✨ Funcionalidades

* **Design Responsivo:** O layout adapta-se perfeitamente a computadores e dispositivos móveis.
* **Navegação Suave (Smooth Scroll):** Transições fluidas ao clicar nos itens do menu de navegação.
* **Saudação Dinâmica:** O cabeçalho lê a hora local da máquina do usuário e exibe automaticamente uma mensagem de "Bom dia", "Boa tarde" ou "Boa noite".
* **Código Semântico e Limpo:** Estruturado com as melhores práticas de desenvolvimento web.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura e semântica de conteúdo.
* **CSS3:** Estilização, paleta de cores baseada na natureza e efeitos visuais (`hover`, transições).
* **JavaScript (Vanilla):** Lógica de interatividade para a página.
